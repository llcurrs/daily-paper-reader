## 摘要
遥感视觉-语言模型推动了地球观测理解的发展，但现有工作大多集中于RGB图像，红外数据中的互补信息尚未得到充分探索。红外图像提供独特的线索，包括热强度结构、物体边界和光照不变场景特征，这些可以丰富传统RGB观测之外的视觉-语言学习。然而，用于遥感视觉-语言建模的大规模RGB-红外-文本数据集仍然缺失。为填补这一空白，我们提出了FusionRS，这是首个专为遥感中双模态视觉-语言学习设计的大规模RGB-红外-文本数据集。FusionRS通过将多种公开的RGB遥感图像转换为红外风格对应图像，构建了对齐的RGB-IR图像对。每个图像对关联有常规场景描述和明确描述红外特有视觉属性同时保留语义内容的红外感知描述。基于FusionRS，我们训练了用于RGB-IR联合理解的双模态视觉-语言基础模型。我们首先训练了CLIP风格模型用于RGB-IR-文本对齐，然后微调生成式VLM用于双模态RGB-IR描述。实验表明，与仅RGB和未使用红外感知训练设置相比，FusionRS改进了RGB-IR对齐、红外到文本检索和双模态描述。消融研究进一步验证了红外感知描述对于加强红外-语言对齐至关重要，突出了模态特定文本监督对于更可扩展的RGB-红外遥感视觉-语言表示学习的重要性。

## Abstract
Remote sensing vision-language models have advanced Earth observation understanding, but most existing work remains centered on RGB imagery, leaving the complementary information in infrared data underexplored. Infrared images provide distinctive cues, including thermal intensity structures, object boundaries, and illumination-invariant scene features, which can enrich visual-language learning beyond conventional RGB observations. However, a large-scale RGB-infrared-text dataset for remote sensing vision-language modeling is still absent. To address this gap, we introduce FusionRS, the first large-scale RGB-infrared-text dataset designed for dual-modal vision-language learning in remote sensing. FusionRS is constructed by translating diverse public RGB remote sensing images into infrared-style counterparts, forming aligned RGB-IR image pairs. Each pair is associated with conventional scene captions and IR-aware captions that explicitly describe infrared-specific visual properties while preserving semantic content. Based on FusionRS, we train dual-modal vision-language foundation models for RGB-IR joint understanding. We first train CLIP-style models for RGB-IR-text alignment, and then fine-tune generative VLMs for dual-modal RGB-IR captioning. Experiments show that FusionRS improves RGB-IR alignment, infrared-to-text retrieval, and dual-modal captioning over RGB-only and non-IR-aware training settings. Ablation studies further verify that IR-aware captions are crucial for strengthening infrared-language alignment, highlighting the importance of modality-specific textual supervision for more scalable RGB-infrared remote sensing vision-language representation learning.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：现有遥感视觉语言模型（如RemoteCLIP、GeoRSCLIP、GeoChat等）几乎全部依赖RGB图像，完全忽略了红外（IR）模态。红外图像能提供热强度结构、物体边界、光照不变特征等互补信息，在弱光、雾霾、阴影等场景下对RGB有重要补充。
- **空白**：当前缺乏大规模、通用型的RGB–红外–文本三元组数据集来支持遥感领域的双模态视觉语言学习。已有的RGB-IR数据集（如LLVIP、M3FD）局限于监控、驾驶等场景，且任务仅限于检测、分割、融合，不涉及视觉语言建模。遥感专用的FireMM-IR仅关注森林火灾监控，不具备通用性。
- **目标**：构建第一个大规模RGB-红外-文本遥感数据集FusionRS（600K三元组），并在此之上训练CLIP风格双模态对齐模型和生成式VLM，实现RGB-IR联合理解。

## 2. 论文提出的方法论

### 2.1 核心思想
将现有RGB遥感图像翻译为红外风格图像，形成对齐的RGB-IR图像对；为每对图像关联原始场景描述和额外生成的**红外感知描述（IR-aware captions）**，从而提供模态特异的语言监督。基于此，训练三模态对比模型（CLIP-style）和生成式VLM。

### 2.2 数据集构建流程
1. **数据源**：从RS5M（488K）、SkyScript（65K）、NWPU（31K）、RSICD（11K）、RSITMD（5K）收集600K RGB遥感图像，按580K/10K/10K划分训练/验证/测试。
2. **RGB→IR翻译**：使用DiffV2IR（扩散模型 checkpoint after_phase_2.ckpt）将每张RGB图像转换为红外风格图像，保存为512×512的.jpg（PIL RGB模式，视觉上呈灰度红外风格但三通道不完全相同）。该过程保留场景布局和物体语义，引入模态特有的视觉特征（强度主导、高对比度、结构轮廓等）。
3. **文本标注**：
   - 保留原始数据集的场景描述（原始描述）。
   - 额外生成**IR-aware captions**：使用Qwen2.5-VL-72B-Instruct通过OpenRouter API，输入RGB图像、红外风格图像、原始文本，生成同时保留场景语义且明确描述红外视觉线索（灰度强度、高对比区域、亮暗结构、纹理减少、物体轮廓等）的描述。共生成49,068条训练IR-aware captions和10,000条测试IR-aware captions。
4. **质量控制**：对原始描述进行规则过滤（去除噪声、非英文、过短/过长、无内容等）；对IR-aware captions进行自动质量检查（去除过短、应答错误、无红外线索等）。最终形成600K RGB-IR-文本三元组（原始描述）和增强的IR-aware子集。

### 2.3 模型训练

#### CLIP-style三模态对比学习
训练目标是在共享嵌入空间中对齐RGB图像、红外图像和文本。给定批量N个三元组，定义RGB图像嵌入 \( \mathbf{z}^r_i \)、IR嵌入 \( \mathbf{z}^v_i \)、文本嵌入 \( \mathbf{z}^c_i \)。对称对比损失公式：
\[
\mathcal{L}_{a,b} = \frac12 \left[ \text{CE}(\mathbf{S}_{a,b}, \mathbf{y}) + \text{CE}(\mathbf{S}_{a,b}^\top, \mathbf{y}) \right],
\]
其中相似度矩阵 \( \mathbf{S}_{a,b} = \tau \cdot \mathbf{Z}_a \mathbf{Z}_b^\top \)，总损失：
\[
\mathcal{L}_{\text{CLIP}} = \frac13 ( \mathcal{L}_{r,c} + \mathcal{L}_{v,c} + \mathcal{L}_{r,v} ),
\]
即RGB-文本对齐、IR-文本对齐、RGB-IR视觉对齐三个损失平均。

#### 红外感知描述增强微调
在CLIP预训练（600K原始描述）之后，使用49,068条IR-aware captions进行微调。设置三种监督方式：
- **A-original**：红外图像+原始描述。
- **B-iraware**：红外图像+IR-aware描述。
- **C-mixed**：混合原始描述+IR-aware描述+RGB-IR图像对。

#### VLM指令微调
使用LoRA（rank=32, alpha=64, dropout=0.05）微调生成式VLM（如Qwen2.5-VL-7B），每个样本包含红外图像、指令（生成描述或回答问题）和目标响应。同样对比A-original、B-iraware、C-mixed三种监督。训练超参数：学习率2e-4，余弦调度，warmup 0.03，bf16，梯度累积8步，有效batch size 8，1 epoch。

## 6. 主要结论与发现

1. **FusionRS显著提升双模态对齐**：在使用原始描述的基础上，CLIP-style模型在IR→text、text→IR、RGB→IR、IR→RGB等所有检索方向上的Recall均高于仅RGB训练的基线。

## 8. 不足与局限

- **红外图像为合成生成**：并非真实传感器捕获，存在模态伪影、强度偏差、不准确的温度响应等问题。这些合成红外图像可能无法完全反映真实红外物理特性（如热辐射、材料响应）。
- **文本噪声**：原始描述和IR-aware captions经过清洗但仍可能包含不完整、弱红外线索或语义错误，影响训练。
- **缺乏真实数据验证**：未在真实RGB-IR配对数据（如DroneVehicle、VEDAI）上测试模型性能，泛化能力未知。
- **任务覆盖有限**：实验仅聚焦检索、描述、VQA，未涉及更广泛的下游应用（如灾害监测、目标检测、变化检测、跨模态地理空间推理）。
- **伦理与安全风险**：合成红外图像不应被用作权威物理测量；模型可能继承源数据集的偏见；需谨慎用于高敏感场景（应急响应、军事分析等）。论文明确指出了这些伦理考虑。
- **计算资源未公开**：训练代价未量化，其他研究者难以评估复现成本。
