## 摘要
我们提出了 Meta-D，这是一种显式利用类别型扫描仪元数据（如 MRI 序列和平面方向）来引导脑肿瘤分析特征提取的架构。我们的目标是通过整合显式元数据来稳定特征表示，从而提高医学图像深度学习流水线的性能。我们首先在二维肿瘤检测中对此进行了评估，其中注入序列（如 T1、T2）和平面（如轴向）元数据可动态调节卷积特征，与仅使用图像的基准模型相比，F1 分数绝对提升了高达 2.62%。由于元数据在数据可用时能为特征提取提供基础，我们假设在数据缺失时，它可以作为一个鲁棒的锚点。我们将此应用于三维缺失模态肿瘤分割。我们的 Transformer Maximizer 利用基于元数据的交叉注意力机制来隔离和路由可用模态，确保网络专注于有效切片。这种有针对性的注意力机制在极端模态匮乏的情况下，将脑肿瘤分割的 Dice 分数提高了多达 5.12%，同时减少了 24.1% 的模型参数。

## Abstract
We present Meta-D, an architecture that explicitly leverages categorical scanner metadata such as MRI sequence and plane orientation to guide feature extraction for brain tumor analysis. We aim to improve the performance of medical image deep learning pipelines by integrating explicit metadata to stabilize feature representations. We first evaluate this in 2D tumor detection, where injecting sequence (e.g., T1, T2) and plane (e.g., axial) metadata dynamically modulates convolutional features, yielding an absolute increase of up to 2.62% in F1-score over image-only baselines. Because metadata grounds feature extraction when data are available, we hypothesize it can serve as a robust anchor when data are missing. We apply this to 3D missing-modality tumor segmentation. Our Transformer Maximizer utilizes metadata-based cross-attention to isolate and route available modalities, ensuring the network focuses on valid slices. This targeted attention improves brain tumor segmentation Dice scores by up to 5.12% under extreme modality scarcity while reducing model parameters by 24.1%.

### 1. 核心问题与研究动机
*   **核心问题**：传统的医疗深度学习模型通常只依赖图像像素，忽略了 DICOM 头文件中的类别型元数据（如 T1、T2 序列信息或轴状、冠状等平面方向）。
*   **研究动机**：
    *   **对比度歧义**：不同 MRI 序列中，某些组织（如脂肪或液体）的亮度可能相似，仅靠图像纹理容易导致模型混淆。
    *   **模态缺失挑战**：在临床实际中，患者往往缺少某些 MRI 序列。现有方法多采用零填充（Zero-padding），这不仅引入了噪声，还增加了无效的计算开销。
    *   **特征不稳定性**：缺乏显式的上下文引导，导致模型在不同扫描条件下提取的特征不够鲁棒。

### 2. 方法论：Meta-D 架构
Meta-D 包含两个针对不同任务设计的核心模块：

*   **2D 肿瘤分类（基于 FiLM）**：
    *   **核心思想**：使用特征线性调制（FiLM）技术。
    *   **技术细节**：通过一个多层感知机（MLP）将元数据字符串（序列+平面）映射为缩放因子 ($\gamma$) 和偏移因子 ($\beta$)。这些参数动态地作用于卷积神经网络（如 ResNet-18）的深层特征图，公式为 $FiLM(x) = \gamma x + \beta$。这迫使网络根据扫描背景重新校准特征提取。
*   **3D 缺失模态分割（Transformer Maximizer, $T_{max}$）**：
    *   **核心思想**：将传统的“空间自注意力”转变为“元数据驱动的交叉注意力”。
    *   **技术细节**：
        *   **查询（Q）**：来自 3D 图像块的嵌入。
        *   **键（K）与值（V）**：来自一个预定义的、固定大小的元数据字典（包含 T1, T1c, T2, FLAIR 四种模态）。
        *   **确定性掩码**：如果某种模态缺失，其对应的注意力权重被强制设为 $-\infty$（Softmax 后变为 0）。这从数学上切断了缺失数据对特征图的干扰。
        *   **复杂度优化**：注意力复杂度从 $O(N^2)$ 降低到 $O(N \cdot M)$（$N$ 为图像块数，$M$ 为模态数，此处 $M=4$）。

### 6. 主要结论与发现
*   **性能提升**：2D 分类 F1 分数提升高达 **2.62%**；3D 分割在极端模态缺失（如仅有 T1）时，Dice 分数提升达 **5.12%**。
*   **效率增强**：相比 MMFormer，Meta-D 的总参数量减少了 **24.1%**，计算量（GFLOPS）减少了 **4.2%**。
*   **特征定位**：可视化显示，注入元数据后，模型的注意力更精准地集中在肿瘤边缘，减少了背景噪声的干扰。

### 7. 优点与亮点
*   **高效性**：通过将空间自注意力改为元数据交叉注意力，显著降低了 Transformer 的计算复杂度。
*   **鲁棒性**：确定性掩码机制解决了缺失模态下的噪声传播问题，使模型在“数据不全”时依然稳定。
*   **即插即用**：FiLM 和 $T_{max}$ 模块可以较容易地集成到现有的医学影像流水线中。

### 8. 不足与局限
*   **硬件限制**：由于显存限制，3D 实验的 Batch size 仅为 1，且裁剪尺寸较小，这可能限制了模型学习全局空间关系的能力。
