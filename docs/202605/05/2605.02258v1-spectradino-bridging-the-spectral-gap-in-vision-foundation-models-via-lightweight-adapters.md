## 摘要
在大规模 RGB 数据上预训练的视觉基础模型（Vision Foundation Models，VFMs）已展示出卓越的表征能力，但其在涵盖近红外（NIR）、短波红外（SWIR）和长波红外（LWIR）的多光谱成像中的适用性仍 largely 未被探索。这些光谱模态提供了对抗恶劣环境条件至关重要的互补感知能力，但相对于以 RGB 为中心的预训练模型存在根本的域间差距。我们提出了 SpectraDINO，这是一种多光谱视觉基础模型，通过轻量级的每模态瓶颈适配器将 DINOv2 ViT 主干扩展至超出可见光范围的模态，从而弥合这一光谱差距，同时保留冻结的 RGB 主干中的丰富表征。我们引入了一种多阶段的教师-学生训练方案，其中冻结的 DINOv2 教师通过余弦蒸馏、对称对比损失、patch 级对齐以及一种新颖的邻域结构保持损失来指导光谱学生模型。该分阶段学习策略实现了强大的跨模态对齐，同时避免了 RGB 先验的灾难性遗忘。我们在具有挑战性的 NIR、SWIR 和 LWIR 基准上，采用广泛使用的融合策略，对 SpectraDINO 在多光谱目标检测和语义分割任务中的表现进行了评估。SpectraDINO 在大多数基准上均取得了领先性能，验证了其作为光谱泛化通用主干网络的有效性。模型的代码和权重可在 https://github.com/Yonsei-STL/SpectraDINO 获取。

## Abstract
Vision Foundation Models (VFMs) pretrained on large-scale RGB data have demonstrated remarkable representation quality, yet their applicability to multispectral imaging spanning Near-Infrared (NIR), Short-Wave Infrared (SWIR), and Long-Wave Infrared (LWIR) remains largely unexplored. These spectral modalities offer complementary sensing capabilities critical for robust perception in adverse conditions, but present a fundamental domain gap relative to RGB-centric pretrained models. We present SpectraDINO, a multispectral VFM that bridges this spectral gap by extending DINOv2 ViT backbones to beyond-visible modalities through lightweight, per-modality bottleneck adapters, while preserving the rich representations of the frozen RGB backbone. We introduce a multi-stage teacher-student training protocol in which a frozen DINOv2 teacher guides a spectral student via cosine distillation, symmetric contrastive loss, patch-level alignment, and a novel neighborhood-structure-preservation loss. This staged curriculum enables strong cross-modal alignment without catastrophic forgetting of RGB priors. We evaluate SpectraDINO on multispectral object detection and semantic segmentation across challenging NIR, SWIR, and LWIR benchmarks using widely adopted fusion strategies. SpectraDINO achieves state-of-the-art performance across most benchmarks, validating its effectiveness as a general-purpose backbone for spectral generalization. The code and weights for model variants are available at https://github.com/Yonsei-STL/SpectraDINO.

## 一、研究问题与背景

- **核心问题**：当前视觉基础模型（Vision Foundation Models, VFMs）主要在大规模 RGB 图像上预训练，表现优异，但迁移到多光谱（包括近红外 NIR、短波红外 SWIR、长波红外 LWIR）任务时性能显著下降。模型在非可见光波段下存在显著的“光谱域间差距”（spectral gap）。
- **研究动机**：现实场景中的感知任务（如夜间监控、自动驾驶、恶劣天气视觉等）依赖红外波段的感知能力，然而当前的基础模型无法自然兼容这些域。  
  因此，作者希望构建一个 **统一、通用的多光谱基础模型**，能在 RGB 与各类红外光谱间实现一致的表示对齐。

## 二、方法论：SpectraDINO 框架与技术细节

### 核心思想
- 在 **冻结的 DINOv2 ViT 主干**上，通过少量可训练模块实现跨光谱迁移和表示对齐。
- 使用 **轻量级瓶颈适配器（bottleneck adapters）** 与 **模态专属输入处理（stems + embeddings）**，保证参数高效与跨模态共享。
- 采用 **教师-学生蒸馏策略**，借助冻结的 DINOv2 RGB 模型作为教师，指导多光谱学生模型学习光谱一致性。

### 架构组成
1. **教师模型**：冻结的 DINOv2（ViT-S/B/L/G）仅处理 RGB 图像，生成稳定语义锚点。
2. **输入层适配**：
   - RGB 使用近似恒等映射的 1×1 卷积。
   - NIR、SWIR、LWIR 各自拥有两层卷积的输入 stem（带 instance normalization），映射单通道红外图像至兼容三通道输入。
3. **轻量瓶颈适配器**：  
   - 每个 transformer block 后插入模态特定的两层 MLP，采用残差连接。  
   - 维度缩减比例 4×，初始输出接近零，避免破坏原有权重。
4. **模态嵌入（embedding）**：为每种光谱建立独立向量，加入至所有 token 表征中，实现统一 backbone 的条件化。
5. **多目标蒸馏与对齐损失**：
   - **余弦蒸馏损失**：保证学生 CLS-token 与教师对应语义对齐。  
   - **对称对比损失（InfoNCE）**：强化跨模态表示一致性（RGB ↔ 红外）。  
   - **Patch-level 对齐**：局部 patch embedding 的对应关系保持。  
   - **邻域结构保持（Neighborhood KL）**：利用 FIFO 队列保存教师特征分布，维持嵌入空间局部拓扑。
6. **三阶段训练流程**：
   - **Stage I**：仅训练适配器、输入层、模态嵌入（冻结 backbone），高蒸馏权重保证稳定迁移。  
   - **Stage II**：引入邻域 KL 损失与教师队列（仍冻结 backbone）。  
   - **Stage III**：部分解冻 50%（或 25%）主干层，重新平衡损失权重，实现跨模态统一优化。

## 六、主要结论与发现

- 单一 SpectraDINO 模型可在 RGB、NIR、SWIR、LWIR 四模态间共享主干，实现统一特征表示。
- 轻量适配器与分阶段蒸馏有效防止灾难性遗忘，强化跨光谱一致性。
- 多个基准上达到或超越最新水平：

  - **LWIR 分割**：SemanticRT、FMB 上 mIoU 全部夺冠。
  - **NIR 分割**：仅用 RGB+NIR 即超越部分四模态模型。

## 七、方法与实验的优点

- **参数高效**：仅添加约 14M 适配器参数（<20% 主干），显著减少存储与计算开销。
- **统一性强**：单模型可处理三种红外模态，无需分别训练。
- **蒸馏与对齐策略创新**：结合局部（patch）、全局（CLS）、拓扑（邻域）三层次目标，兼顾语义一致与结构稳定。
- **训练策略稳健**：分阶段引入不同损失，避免初期不稳定或表示坍缩。
- **全面实验验证**：任务覆盖检测、分割两个视角，基准充分。

## 八、不足与局限

- **数据局限**：仅有二模态（RGB-IR）配对数据，缺乏多红外同时观察的三模态或四模态样本。
- **模态独立性问题**：模型通过 RGB 作为对齐中介，各红外间无直接约束，可能限制深度红外间迁移。
- **采样策略简单**：使用固定轮转（round-robin），未考虑模态间难度差异；潜在可改进为自适应采样。
- **计算资源虽中等但有限**：多模态训练可能在更大规模数据上需更强 GPU 支撑。
- **性能偏差**：部分 LWIR 数据集提升有限，显示对发射型红外信号仍存在表征不足。
- **应用限制**：当前设计仅涵盖静态图像任务，未验证视频或时序多光谱场景。
