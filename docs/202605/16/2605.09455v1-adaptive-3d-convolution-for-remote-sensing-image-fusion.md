## 摘要
遥感影像融合的目标是从一个具有有限光谱信息的高分辨率图像和一个具有丰富光谱数据的低分辨率图像中生成高分辨率的多光谱或高光谱图像。近年来，深度学习（DL）技术在该领域表现出显著的有效性。大多数基于深度学习的方法将图像融合视为一个二维问题，通过将光谱信息编码到特征图通道中来处理。然而，我们的研究表明，这种策略会引入明显的光谱失真。相比之下，一些方法将光谱数据视为一个附加维度，采用标准三维卷积以保留光谱信息。然而，在标准三维卷积层中，同一组卷积核被应用于所有输入区域，这在图像融合任务中被我们发现并非最优。此外，标准三维卷积需要大量计算资源。为应对这些挑战，我们提出了一种新的卷积范式——用于遥感影像融合的自适应三维卷积（Adaptive 3D Convolution，Ada3D）。Ada3D 为每个输入体素分配一组独特的三维卷积核，从而能够捕捉细粒度的细节。这些自适应卷积核通过两步过程生成：（i）分别从空间和光谱图像源中提取空间卷积核和光谱卷积核；（ii）将这两类卷积核结合形成具备内容感知能力的三维卷积核，以高效整合空间与光谱信息。此外，我们引入了自适应偏置，以提升体素级卷积效果。进一步地，我们结合组卷积技术以降低计算复杂度。由此，Ada3D 以高效的方式实现了全面的自适应性。对五个数据集的评估结果表明，我们的方法达到了当前最先进（SOTA）的性能，突显了 Ada3D 的优越性。代码可在 https://github.com/PSRben/Ada3D 获取。

## Abstract
Remote sensing image fusion aims to create a high-resolution multi/hyper-spectral image from a high-resolution image with limited spectral information and a low-resolution image with abundant spectral data. Recently, deep learning (DL) techniques have shown significant effectiveness in this area. Most DL-based methods approach image fusion as a 2D problem by encoding spectral information into feature map channels. However, our research suggests that this strategy introduces notable spectral distortions. In contrast, some methods consider spectral data as an additional dimension, utilizing standard 3D convolutions to preserve spectral information. Nevertheless, in a standard 3D convolutional layer, the same set of kernels is applied across all input regions, which we have found to be sub-optimal for image fusion. Furthermore, standard 3D convolutions necessitate substantial computational resources. To address these challenges, we propose a novel convolutional paradigm called Adaptive 3D Convolution (Ada3D) for remote sensing image fusion. Ada3D applies a unique set of 3D kernels to each input voxel, enabling the capture of fine-grained details. These adaptive kernels are generated through a two-step process: (i) spatial and spectral kernels are derived from their respective image sources; (ii) these two types of kernels are then combined to form content-aware 3D kernels that effectively integrate spatial and spectral information. Additionally, adaptive biases are introduced to enhance the convolutional outcome at the voxel level. Furthermore, we incorporate the group convolution technique to reduce computational complexity. As a result, Ada3D offers full adaptivity in an efficient manner. Evaluation results across five datasets demonstrate that our method achieves SOTA performance, underscoring the superiority of Ada3D. The code is available at https://github.com/PSRben/Ada3D.

## 一、研究背景与问题动机

- **研究主题**：遥感图像融合（Remote Sensing Image Fusion），旨在将高空间分辨率但光谱信息有限的影像与低空间分辨率但光谱丰富的影像结合，生成高分辨率的多光谱或高光谱图像。  
- **背景问题**：
  - 卫星传感器受硬件限制，难以同时获取高空间与高光谱图像。
  - 传统方法（如成分替代法、基于多分辨率分析、变分优化）在光谱或空间一致性方面存在明显取舍。
  - 深度学习方法虽然取得突破，但多数将融合问题视作二维问题，将光谱信息编码至通道维度，导致**光谱失真（spectral distortion）**。
  - 标准三维卷积（Conv3D）能保留光谱信息，但存在**核共享导致的内容敏感度不足**与**计算代价过高**的问题。
- **研究动机**：开发一种既能保持光谱保真，又能动态适应输入内容、同时具备高计算效率的三维卷积机制。

## 二、方法论与技术细节

### 1. 核心思想
提出**自适应三维卷积（Adaptive 3D Convolution，Ada3D）**，为每个体素（voxel）生成独特的三维卷积核与偏置，从而实现空间-光谱双自适应融合，并配合组卷积降低计算开销。

### 2. 数学与理论分析
- **定理与推导**：
  - 通过线性代数形式证明将图像融合建模为二维问题会引起**rank(A) < L**的光谱信息丢失。
  - 进一步证明标准卷积共享核的方式在非齐次方程组意义下是次优的，无法满足局部目标输出。
- **Remark 1**：二维建模会造成光谱维度降秩损失。  
- **Remark 2**：标准卷积的核共享机制限制了其对场景内容的适应性。

### 3. 模块结构
包括以下关键组件（见 Fig. 5）：

- **空间卷积分支（Spatial branch）**：从高分辨率 PAN 图像生成空间特征 (Fa)。  
- **光谱卷积分支（Spectral branch）**：从低分辨率多/高光谱图像生成光谱特征 (Fb)。  
- **自适应核生成器（Kernel Generator）**：
  - 空间核自 PAN 图像提取；
  - 光谱核自 LRHS/LRMS 图像提取；
  - 使用张量点积融合两类核生成内容相关的 3D 核 K。  
- **自适应偏置生成器（Bias Generator）**：
  - 从空间、光谱和通道三个方向生成偏置 Da、Db、Dc；
  - 综合得到体素级偏置 D。  
- **组卷积与高效实现**：
  - 在通道分组下生成 C 组 3D 卷积核；
  - FLOPs 降至与 2D 方法相近（≈2.64G FLOPs）。

### 4. 公式化表达（简要）
Ada3D 操作公式：
\[
\tilde{F}(·, i) = D(·, i) + \sum_{j \in \Omega(i)} K(p_i,·,p_i-p_j) ⊙ F(·,j)
\]
其中每个体素 i 使用独立 3D 核 `K` 与偏置 `D`。

### 5. 损失函数
总损失由 **L1** 与 **ERGAS** 组成：
\[
\text{Loss} = L_{1} + λ_{ergas}L_{ergas}
\]
ERGAS 用于衡量光谱保真度，λ 控制权重。

## 六、主要结论与发现

1. **二维建模引发光谱失真，三维建模显著改善光谱保真**。  
2. **标准卷积的核共享机制次优**，而自适应核在数学上可拟合目标分布。  
3. **Ada3D 实现三维自适应融合**，在相近参数量和 FLOPs 条件下：
   - 在 WDC、Botswana、Pavia 上分别实现 +0.5~1.0 dB PSNR 提升；
   - 在 WV3 任务中 QNR = 0.9554（为所有方法最高）；
   - 在 CAVE 数据集上 HISR 任务达 PSNR 52.018 dB，领先所有基线。  
4. **推理效率优越**：耗时 17.8ms、能耗 4.42J，优于其他 3D 卷积模型。  
5. 证明 Ada3D 具备良好的泛化，可作为**可插拔模块**嵌入 PNN、FusionNet、U2Net 等网络。
