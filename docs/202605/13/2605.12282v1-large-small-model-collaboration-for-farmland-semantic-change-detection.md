## 摘要
农田语义变化检测（Farmland Semantic Change Detection，SCD）在耕地保护中至关重要，但现有的基准和模型在精细化的农田转换监测方面仍显不足。目前的数据集通常缺乏专门的“从–到”标注，而视觉变化检测模型易受到作物轮作、季节变化和光照差异引起的物候伪变化干扰。为解决这些挑战，我们构建了HZNU-FCD，一个具有统一的五类农田到非农田标注协议的大规模精细化农田SCD基准。该数据集包含4588组双时相图像对，并提供用于实际耕地保护的像素级标签。在此基准上，我们提出了一种大小模型协同的SCD框架，将任务驱动的小型视觉模型与冻结的大型视觉–语言模型集成。小模型Fine-grained Difference-aware Mamba（FD-Mamba）学习密集的变化表示，以实现边界保持与小区域定位；大型模型分支Cross-modal Logical Arbitration（CMLA）引入基于CLIP的文本先验，通过提示引导的语义裁决抑制伪变化。为实现有效协作，我们设计了一种硬区域协同训练策略，仅在低置信度像素上监督CMLA的语义得分图。实验结果表明，本方法在HZNU-FCD上仅使用6.65M可训练参数即可达到97.63%的F1、96.32%的IoU和96.35%的SCD_IoU_mean。与利用视觉–语言信息进行变化检测的多模态ChangeCLIP-ViT相比，我们的方法在HZNU-FCD上的F1提升了10.19个百分点。此外，在LEVIR-CD上获得91.43%的F1和84.21%的IoU，在WHU-CD上获得93.85%的F1和88.41%的IoU，展示出强大的鲁棒性和泛化能力。代码可在 https://github.com/Lovelymili/FD-Mamba 获取。

## Abstract
Farmland Semantic Change Detection (SCD) is essential for cultivated land protection, yet existing benchmarks and models remain insufficient for fine-grained farmland conversion monitoring. Current datasets often lack dedicated "from-to" annotations, while visual change detection models are easily disturbed by phenology-induced pseudo-changes caused by crop rotation, seasonal variation, and illumination differences. To address these challenges, we construct HZNU-FCD, a large-scale fine-grained farmland SCD benchmark with a unified five-class farmland-to-non-farmland annotation protocol. It contains 4,588 bitemporal image pairs with pixel-level labels for practical farmland protection. Based on this benchmark, we propose a large-small collaborative SCD framework that integrates a task-driven small visual model with a frozen large vision-language model. The small model, Fine-grained Difference-aware Mamba (FD-Mamba), learns dense change representations for boundary preservation and small-region localization. The large-model pathway, Cross-modal Logical Arbitration (CMLA), introduces CLIP-based textual priors for prompt-guided semantic arbitration and pseudo-change suppression. To enable effective collaboration, we design a hard-region co-training strategy that supervises the CMLA semantic score map only on low-confidence pixels. Experiments show that our method achieves 97.63% F1, 96.32% IoU, and 96.35% SCD_IoU_mean on HZNU-FCD with only 6.65M trainable parameters. Compared with the multimodal ChangeCLIP-ViT, which leverages vision-language information for change detection, our method improves F1 by 10.19 percentage points on HZNU-FCD. It also achieves 91.43% F1 and 84.21% IoU on LEVIR-CD, and 93.85% F1 and 88.41% IoU on WHU-CD, demonstrating strong robustness and generalization. The code is available at https://github.com/Lovelymili/FD-Mamba.

## 二、方法论：框架与关键技术

### 1. 总体框架

提出一种**大—小模型协同框架**，由：
- **小模型（FD-Mamba）**：高效视觉分支，用于学习密集变化表示；
- **大模型（CMLA）**：冻结的CLIP文本编码分支，用于提供语义逻辑判别与伪变化抑制。

两者通过**“硬区域协同训练（hard-region co-training）”**策略交互优化，只在主模型预测置信度低的像素上引入语义校正监督。

### 2. 小模型：Fine-grained Difference-aware Mamba (FD-Mamba)

FD-Mamba建立在**Mamba状态空间模型（SSM）**基础上，结合局部卷积操作与长程状态建模，实现高分辨率变化提取。  
主要组成模块：

- **PDA（Physical Difference Anchor）**：计算双时相特征的绝对差值 + 最大响应，保留几何结构。  
- **MSDE（Multi-scale Spatial Difference Excavation）**：多尺度卷积提取变化特征，兼顾细小道路与大区域建筑。  
- **DPSE（Dual-Path Difference Purification and Saliency Enhancement）**：利用SE + CBAM双通道注意力去除光照和季节伪变化。  
- **DRSA（Dimensionality Reduction and Redundancy Suppression Aggregation）**：融合原时相特征，降维抑制冗余背景。  
- **Conv-Mamba重建单元**：结合卷积与Mamba机制进行全局上下文优化。

这些模块的组合，使模型在农田中能保持细长边界和小面积变化检测能力。

### 3. 大模型：Cross-modal Logical Arbitration (CMLA)

- 基于**冻结的 CLIP 文本编码器**，通过文本提示构建语义原型。  
  - **Brief Prompt**：任务类别（如“farmland change to building”）。  
  - **Input Prompt**：场景及干扰因素（如“Satellite image, ignore shadow”）。
- **操作流程**：  
  1. 将Prompt经文本编码器生成类别原型与上下文偏置。  
  2. 将视觉特征投影到CLIP语义空间，计算像素级相似度（Einsum操作）。  
  3. 得到语义分数图并生成像素级**语义门控（semantic gate）**，用于强化真实变化、抑制伪变化。  
  4. 在训练阶段，仅对**置信度低的像素**施加强化监督（Hard-region Loss）。

### 4. 协同训练策略

总损失定义为：
\[
L_{total} = L_{main} + \lambda L_{aux}
\]
- **主损失**：交叉熵 + Dice Loss（处理类别不均衡）。  
- **辅助损失**：基于CMLA语义图，仅在低置信度区域（置信度 < 0.8）进行辅助学习。

通过这种分区监督，实现语义分支与视觉分支的互补与稳健协作。
