## 摘要
在高光谱图像（HSI）分类任务中，每个像素都被划分为特定的地物类别或材料。卷积神经网络（CNN）和Transformer广泛用于HSI分类中的局部和非局部特征提取。近年来的研究利用多尺度视觉Transformer（ViT）增强了光谱特征的捕获，并取得了较为理想的结果。然而，现有多数方法在空间-光谱信息的有效联合利用以及特征在传播过程中的跨层信息保持方面仍面临挑战。为了解决这些问题，我们提出了一种用于HSI分类的协同CNN-Transformer网络，结合了池化注意力融合机制，以协同方式利用CNN和ViT分别处理空间与光谱特征。具体而言，我们设计了一个双支路特征提取（TBFE）模块，采用3D和2D卷积并行操作，从HSI中全面提取光谱与空间特征。同时，设计了一种混合池化注意力（HPA）模块，用于聚合空间注意力。此外，引入级联Transformer编码器以进行全局光谱特征提取，并设计了一个简单而高效的跨层特征融合（CFF）模块，以减少前序网络层中关键信息的损失。我们在多个具有代表性的数据集上进行了大量实验，结果表明所提出的方法相比现有先进技术具有更优的性能。代码可在 https://github.com/chenpeng052/SCT-Net.git 获取。

## Abstract
In the hyperspectral image (HSI) classification task, each pixel is categorized into a specific land-cover category or material. Convolutional neural networks (CNNs) and transformers have been widely used to extract local and non-local features in HSI classification. Recent works have utilized a multi-scale vision transformer (ViT) to enhance spectral feature capture and yield promising results. However, most existing methods still face challenges in the effective joint use of spatial-spectral information and in preserving information across layers during the propagation process. To address these issues, we propose a synergistic CNN-Transformer network with pooling attention fusion for HSI classification, which collaboratively utilizes CNNs and ViT to process spatial and spectral features separately. Specifically, we propose a Twin-Branch Feature Extraction (TBFE) module, which employs 3D and 2D convolution in parallel to comprehensively extract spectral and spatial features from HSI. A hybrid pooling attention (HPA) module is designed to aggregate spatial attention. Moreover, a cascade transformer encoder is employed for global spectral feature extraction, and a simple yet efficient cross-layer feature fusion (CFF) module is designed to reduce the loss of crucial information in the previous network layers. Extensive experiments are conducted on several representative datasets to demonstrate the superior performance of our proposed method compared to the state-of-the-art works. Code is available at https://github.com/chenpeng052/SCT-Net.git.

- **研究背景：**  
  高光谱图像（HSI）具有上百个连续的光谱通道，可同时反映空间结构与光谱属性，被广泛应用于土地覆盖检测、精准农业、城市规划等任务。  
  传统机器学习方法（SVM、KNN、RF 等）擅长处理高维特征，但难以充分捕捉空间域与光谱域的复杂关联。  
  深度学习（CNN、Transformer、RNN、GAN 等）在 HSI 分类中表现突出，但仍存在以下常见问题：
  - **空间-光谱特征融合不足：** CNN 强于局部空间特征提取，但难以捕获长程光谱依赖；Transformer 拥有全局建模能力，却对浅层局部特征敏感度较低。  
  - **层间信息流失问题：** 编码器层数增加导致不同层特征无法充分融合，信息传播过程中发生损失。

- **核心研究问题：**  
  如何在高光谱图像分类中**有效融合局部空间特征与全局光谱特征**，并同时**保持跨层信息完整性**，实现更高分类精度与更强的特征表达能力。

## 2. 方法论与关键技术

### 2.1 整体框架
作者提出了一个**协同 CNN-Transformer 网络（SCT-Net）**，通过**“双分支特征提取 + 混合池化注意力 + 跨层特征融合”**的多模块结构，实现空间与光谱信息的协同学习。

### 2.2 核心模块说明

1. **Twin-Branch Feature Extraction (TBFE) 模块：**
   - **思想：** 在特征提取初期并行使用 3D 与 2D 卷积。
   - **结构：**
     - 一支为 3D 卷积分支（1×1×3 卷积），专注于光谱维度建模；
     - 另一支为 2D 卷积分支（3×3 卷积），捕捉空间关联；
     - 最后通过拼接（Concat）操作融合两个分支的输出。
   - **创新性：** 同时利用 3D CNN 的光谱依赖捕捉与 2D CNN 的空间特征提取能力，实现轻量化的互补特征学习。

2. **Hybrid Pooling Attention (HPA) 模块：**
   - **功能：** 在 CNN 输出后引入高效的空间注意机制，兼顾全局性与局部性。
   - **机制：**
     - 对输入特征图进行分组，分别在水平与垂直方向上施加平均池化（GAP）与最大池化（GMP）；
     - 将两个分支的注意力特性融合，通过 1×1 卷积与 Sigmoid 激活函数生成空间-通道注意权重；
     - 通过加权矩阵乘法进行**跨空间维度交互学习**，提升像素级分类能力。

3. **Cascade Transformer Encoder：**
   - **作用：** 在注意力加权的基础上使用多层 Transformer 编码器，用于全局光谱依赖建模；
   - **输入：** 由 HPA 层输出的序列化 token；
   - **机制：** 加入可学习分类标记（[CLS] token），结合位置嵌入进行全局特征学习；
   - **核心公式：** 使用标准自注意力机制 `Attention(Q,K,V) = softmax(QK^T/√d_k)V`；
   - **多头机制（MSA）：** 结合多个 attention head 并行建模多尺度特征关系。

4. **Cross-layer Feature Fusion (CFF) 模块：**
   - **思想：** 提升层与层之间的信息传递，防止梯度衰减与特征丢失；
   - **实现：** 在多层 Transformer 编码器之间添加可学习融合权重 `w`，对不同层输出进行拼接与加权整合；
   - **优势：** 兼顾深层与浅层特征的互补表征能力。
