## 摘要
多模态遥感影像的语义分割在土地利用/土地覆被（LULC）制图、环境监测和精密地球观测中发挥着关键作用。当前的多模态方法主要关注于整合互补的视觉模态，却忽略了非视觉文本数据的引入——这是一种可以弥合视觉模式与真实世界概念之间语义差距的丰富知识来源。为解决这一局限，我们提出了TSMNet，一种文本监督的多模态开词汇语义分割网络，通过将文本监督与视觉表征协同融合，实现开词汇语义分割。与传统的多模态分割框架不同，TSMNet引入双分支文本编码器，从多样的文本数据中同时提取场景级语义和目标级标签信息，从而实现动态跨模态融合。这些文本派生特征通过所提出的文本引导视觉语义融合模块与视觉嵌入进行动态交互，实现具有领域感知的特征优化与具有人类可解释性的决策。为验证我们的方法，我们创新性地构建了两个新的多模态数据集，并进行了大量实验，对所提方法与其他先进（SOTA）语义分割模型进行了全面比较。实验结果表明，TSMNet在保持优异分割精度的同时，展现出在多样地理与传感器特定场景下的强泛化能力。本研究为可解释的遥感分析建立了一种新的范式，表明文本知识的融入显著增强了模型的可泛化性。源代码将发布于 https://github.com/yeyuanxin110/TSMNet

## Abstract
Semantic segmentation of multi-modal remote sensing imagery plays a pivotal role in land use/land cover (LULC) mapping, environmental monitoring, and precision earth observation. Current multi-modal approaches mainly focus on integrating complementary visual modalities, yet neglect the incorporating of non-visual textual data - a rich source of knowledge that can bridge semantic gaps between visual patterns and real-world concepts. To address this limitation, we propose TSMNet, a text supervised multi-modal open vocabulary semantic segmentation network that synergistically integrates textual supervision with visual representation for open-vocabulary semantic segmentation. Unlike conventional multi-modal segmentation frameworks, TSMNet introduces a dual-branch text encoder to extract both scene-level semantic and object-level label information from various textual data, enabling dynamic cross-modal fusion. These text-derived features dynamically interact with visual embeddings through the proposed text-guided visual semantic fusion module, enabling domain-aware feature refinement and human-interpretable decision-making. To verify our method, we innovatively construct two new multi-modal datasets, and carry out extensive experiments to make a comprehensive comparison between the proposed method and other state-of-the-art (SOTA) semantic segmentation models. Results demonstrate that TSMNet achieves superior segmentation accuracy while exhibiting robust generalization capabilities across diverse geographical and sensor-specific scenarios. This work establishes a new paradigm for explainable remote sensing analysis, demonstrating that textual knowledge integration significantly enhances model generalizability. The source code will be available at https://github.com/yeyuanxin110/TSMNet

## 一、研究动机与背景

- **核心问题**：传统的多模态遥感影像语义分割主要整合光学影像与SAR等视觉模态，但**忽略了非视觉模态——文本数据**的潜在价值。缺乏文本语义信息使得模型难以理解复杂地表语义，且无法灵活应对开放词汇（open-vocabulary）场景。
- **研究背景**：
  - 多模态遥感数据（光学、SAR、LiDAR 等）丰富而异质，但视觉模态融合仍受限于模态差异和语义对齐问题。
  - 开词汇语义分割（OVSS）在自然图像领域显示出强大潜力，可通过语言描述识别未见类别。
  - 视觉语言模型（如 CLIP、BLIP、SegCLIP）具备跨模态理解能力，却缺乏针对遥感图像的像素级语义分割适配机制。
- **研究目标**：构建一个同时具备视觉融合与文本理解能力的开放词汇语义分割框架，实现**跨模态语义对齐与细粒度地物识别**。

## 二、方法论与技术框架

### 1. 基本思想
提出 **TSMNet（Text-Supervised Multi-modal Network）**，通过文本监督将遥感视觉特征（光学+SAR）与语言表示统一到共享的语义空间，实现开放词汇语义分割。

### 2. 网络总体结构
网络分为三大阶段：
1. **多模态影像特征融合阶段**（Multi-modal Image Feature Fusion）
2. **目标级（对象级）图文特征融合阶段**（Object-level Label Feature Fusion）
3. **场景级语义图文特征融合阶段**（Scene-level Semantic Feature Fusion）

### 3. 核心模块
- **多模态影像特征融合网络（MFFN）**
  - 使用 ViT 作为光学与SAR双分支主干。
  - 引入两种关键模块：
    - **MFRM（Feature Rectification Module）**：动态校正通道与空间权重以对齐跨模态特征。
    - **MFFM（Feature Fusion Module）**：通过跨模态注意力机制在多层特征中实现语义交互与融合。
- **目标级特征融合模块**
  - 基于 CLIP 文本编码器生成模板化提示（如 “a photo of a [CLS]”）。
  - 使用上下文感知策略增强文本特征，通过残差调节参数 γ 保留语义一致性。
  - 通过余弦相似度生成像素级图文匹配得分图（score map），再与FPN结构结合提升分割结果。
- **场景级特征融合模块**
  - 先对齐（alignment）后融合（fusion），采用对比学习（InfoNCE loss）优化图文特征一致性。
  - 基于跨模态注意生成多尺度文本特征，与视觉特征层层交互，形成全局语义引导。

### 4. 损失函数设计
结合四种损失：
- 对象级标签损失（交叉熵）
- 场景级语义对比损失（InfoNCE）
- 语义分割任务损失（cross-entropy + Dice loss）
- 总损失为上述四项加权求和。

### 5. 推理阶段
输入文本类别→编码生成目标嵌入→计算与各像素语义特征的余弦相似度→最大相似值对应像素类别，实现开放词汇分割。

## 三、实验设计

### 1. 数据集
作者构建了 **两个新的多模态视觉语言数据集**：
- **SWJTU-Vision-Language Dataset**
  - 光学 + SAR，空间分辨率3m，共2712对影像。
  - 覆盖北京、济南、上海、天津等代表性城市地区。

- **YESeg-OPT-SAR Dataset**
  - 光学 + SAR，空间分辨率0.5m，共2231对影像。
  - 八类像素级标注，包含详细文本说明。

## 六、主要结论与发现

1. **TSMNet 性能最优**  
   - 在 SWJTU 数据集上：mIoU = 48.83%，OA = 69.73%；
   - 在 YESeg 数据集上：mIoU = 66.69%，OA = 85.33%，均超越次优 DenseCLIP。
2. **文本融合显著提升分割能力**：模型可准确解释语义相似类别（如裸地与低植被）；
3. **开放词汇能力**：支持用户通过任意文本类别描述自定义地物，无需预定义类集；
4. **语义一致性与解释性增强**：跨模态注意结合文本引导，提高了模型可解释性。
