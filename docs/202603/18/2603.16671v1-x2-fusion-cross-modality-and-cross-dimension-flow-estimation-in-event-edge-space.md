## 摘要
估计稠密 2D 光流和 3D 场景流对于动态场景理解至关重要。近期的工作结合了图像、激光雷达（LiDAR）和事件数据来联合预测 2D 和 3D 运动，但大多数方法在相互独立的异构特征空间中运行。由于缺乏所有模态都能对齐的共享潜空间，这些系统依赖于多个特定模态的模块，导致跨传感器失配问题无法解决，并使融合变得异常复杂。事件相机天然地提供时空边缘信号，我们可以将其视为一种内在边缘场，用以锚定统一的潜表示，称之为“事件边缘空间”（Event Edge Space）。基于这一想法，我们提出了 $x^2$-Fusion，它将多模态融合重新定义为表示统一：由事件衍生的时空边缘定义了一个以边缘为中心的同构空间，而图像和 LiDAR 特征在该共享表示中被显式对齐。在该空间内，我们执行可靠性感知自适应融合，以估计模态可靠性并在性能退化的情况下强调稳定线索。我们进一步采用跨维度对比学习，将 2D 光流与 3D 场景流紧密耦合。在合成和真实基准测试上的大量实验表明，$x^2$-Fusion 在标准条件下达到了最先进的精度，并在挑战性场景中实现了显著提升。

## Abstract
Estimating dense 2D optical flow and 3D scene flow is essential for dynamic scene understanding. Recent work combines images, LiDAR, and event data to jointly predict 2D and 3D motion, yet most approaches operate in separate heterogeneous feature spaces. Without a shared latent space that all modalities can align to, these systems rely on multiple modality-specific blocks, leaving cross-sensor mismatches unresolved and making fusion unnecessarily complex.Event cameras naturally provide a spatiotemporal edge signal, which we can treat as an intrinsic edge field to anchor a unified latent representation, termed the Event Edge Space. Building on this idea, we introduce $x^2$-Fusion, which reframes multimodal fusion as representation unification: event-derived spatiotemporal edges define an edge-centric homogeneous space, and image and LiDAR features are explicitly aligned in this shared representation.Within this space, we perform reliability-aware adaptive fusion to estimate modality reliability and emphasize stable cues under degradation. We further employ cross-dimension contrast learning to tightly couple 2D optical flow with 3D scene flow. Extensive experiments on both synthetic and real benchmarks show that $x^2$-Fusion achieves state-of-the-art accuracy under standard conditions and delivers substantial improvements in challenging scenarios.

#### 2. 方法论
论文提出了 **$x^2$-Fusion** 框架，其核心是将多模态融合重构为“表示统一”问题：
*   **事件边缘空间 (Event Edge Space, EES)**：利用事件相机对移动边缘极其敏感的特性，构建一个以边缘为中心的同构潜空间。
*   **关键技术细节**：
    1.  **事件边缘编码器预训练**：首先通过自监督学习预训练一个事件编码器，预测未来的边缘强度，从而提取稳定的运动边缘特征。随后将其冻结，作为“边缘原型（Edge Prototypes）”。
    2.  **边缘锚定对称正则化**：将图像和 LiDAR 特征投影到 EES 中，通过 $\ell_1$ 距离损失函数强制它们与冻结的事件边缘特征对齐，实现特征层面的同构化。
    3.  **可靠性感知自适应融合**：在同构空间内，通过运动流（全局一致性）和空间流（局部结构一致性）估计各模态的可靠性权重，并利用交叉注意力 Transformer 进行特征增强。
    4.  **跨维度对比学习 (CCL)**：
        *   **Pull（拉近）**：增强 2D 和 3D 运动向量在时间上的相干性。
        *   **Push（推开）**：通过变分编码最小化 2D 与 3D 特征间的互信息，鼓励它们学习互补的内部表示。

#### 6. 主要结论与发现
*   **统一空间的优越性**：将异构特征对齐到“事件边缘空间”比直接进行多模态交互更有效，能显著降低融合难度。
*   **鲁棒性提升**：在图像失效（如极暗环境）时，系统能自动依赖事件和 LiDAR 的边缘信息；在 LiDAR 稀疏时，图像和事件提供的结构线索能弥补几何缺失。
*   **跨维度协同**：联合训练 2D 和 3D 任务并引入对比学习约束，比独立训练两个任务的效果更好。

#### 7. 优点
*   **架构简洁**：通过同构空间简化了多模态交互逻辑，避免了繁琐的成对对齐模块。
*   **边缘锚定思路新颖**：巧妙地利用事件相机作为图像（2D 像素对齐）和 LiDAR（稀疏结构相似）之间的桥梁。
*   **极端环境表现强劲**：在传感器退化实验中，相比基准方法（如 RPEFlow）在 ACC 1px 等指标上有高达 30% 以上的绝对提升。

#### 8. 不足与局限
*   **模态依赖**：虽然 EES 理论上是模态无关的，但目前高度依赖事件相机作为“教师”来定义边缘空间。如果事件相机本身发生故障（如过热噪声），系统的稳定性尚未讨论。
*   **计算复杂度**：虽然融合模块轻量化，但处理 3D 稀疏卷积和 Transformer 交叉注意力在嵌入式设备上的实时性表现仍需进一步验证。
*   **应用限制**：目前主要针对流估计任务，对于其他 3D 感知任务（如目标检测、语义分割）的泛化能力有待探索。
